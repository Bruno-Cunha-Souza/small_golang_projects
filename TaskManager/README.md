# Task Manager

## Commands

### Add Task

``` Go
go run cmd/main.go add "Task Title" "Task Description" "0000-00-00"
```

### List Tasks

``` Go
go run cmd/main.go list
```

### Update Task Status

``` Go
go run cmd/main.go update [id] [status]
```

### Delete Task

``` Go
go run cmd/main.go delete [id]
```
